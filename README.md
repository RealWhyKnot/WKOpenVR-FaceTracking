# WKOpenVR Face Tracking

Face Tracking is the WKOpenVR module for face and eye tracking in VRChat through the WKOpenVR face module host.

Download the latest `WKOpenVR-FaceTracking-v...-Setup.exe` from Releases and run it. The installer installs WKOpenVR, registers the SteamVR driver, and enables Face Tracking. Other WKOpenVR modules can be enabled from the Modules tab or by running another module installer.

## Source

This repository is the installer and release page for the module. Source code lives in the WKOpenVR source repository:

https://github.com/RealWhyKnot/WKOpenVR/tree/main/modules/facetracking

## Upstream
